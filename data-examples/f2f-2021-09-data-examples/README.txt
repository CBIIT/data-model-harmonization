This directory holds harmonized data examples generated for the September 2021 CRDC-H Pilot Demonstration. 

Files beginning with 'd1' are part of Demonstrator 1, and those beginning with 'd2' are part of Demonstrator 2.

Documentation about these Demonstrators and data examples can be found in September F2F Slide Deck, starting here: https://docs.google.com/presentation/d/1bw2iY7yoyCAGCKYu21fcSHo7rWXZgbeVac2joTShpPg/edit#slide=id.gef28fd37cb_0_588
